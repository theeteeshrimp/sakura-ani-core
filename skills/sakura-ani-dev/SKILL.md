# Sakura Ani Dev

## Build Loop
1. Define one feature slice.
2. Implement smallest working version.
3. Test with 3 sample conversations.
4. Commit with clear message.
5. Repeat.

## Priority Order
1. Chat API
2. Persona controls
3. Memory
4. Tool integrations
5. Media/image extras

## Rule
Keep each change shippable and demoable.
