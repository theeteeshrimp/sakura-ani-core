# Product Notes

## Core MVP
- Chat with anime persona
- Configurable persona profile
- Memory toggle (session vs persistent)
- Tool action stubs (search, summarize, schedule)

## Personality Controls
- `vibe`: chill | energetic | tsundere-lite | mentor
- `emoji_level`: low | medium | high
- `directness`: soft | balanced | blunt

## Safety
- No manipulative behavior
- Respect user intent and privacy
- External actions require explicit user confirmation
